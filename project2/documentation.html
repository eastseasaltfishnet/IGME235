Special Features

This project includes the following features:

  • Search Functionality: Allows users to search for random dog images or specific breeds.
  • Favorites: Users can favorite images and view them later.
  • Persistent Search: The last searched term is saved and restored on page reload.
  • All the breed button: Speding a lot of time on the breed button to make this work easier for me
